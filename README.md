## ✎ᝰ. RUNNING

- Clone Repository

```bash
git clone https://github.com/TestnetMaster29/monadscore-auto.git
cd monadscore-auto
```

- Install Dependency

```bash
npm install
```

- Setup config in .env

```bash
nano .env
```

- Setup input value

* proxy: http://user:pass@ip:port

```bash
nano proxy.txt
```

- private key

```bash
nano privateKeys.txt
```

- Run the script

```bash
node main.js
```
